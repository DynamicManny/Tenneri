SENTINEL: SOC Training Simulation

The Problem

Cybersecurity training programs need analysts to experience realistic SOC environments before they encounter one on the job. Traditional approaches depend on expensive simulation platforms, dedicated lab infrastructure, or external tools that add per-seat costs to every cohort and create long-term dependency on vendor pricing decisions.

The Approach

SENTINEL is a full SOC dashboard built entirely in HTML, CSS, and JavaScript and deployed as a native LMS content page. No external platforms, no integration work, no additional licensing. It runs on the system the institution already operates.

What it includes
  • Live alert queue with triage workflow, severity classification, and escalation logic
  • Real-time network traffic chart with sliding UTC timestamps and animated data simulation
  • Threat category breakdown with animated detection coverage indicators across six categories
  • Analyst workload management with case redistribution and narrative-driven team events
  • Suspicious connection table, IOC confidence scoring, and critical asset status panel
  • All data layers interconnected: resolved cases, open alerts, and analyst load move together
  • New analyst joins mid-session and cases redistribute from the highest-load analyst automatically
The result

A single build replaces a per-seat simulation platform subscription for the full duration of the program. The environment is owned outright by the institution, version-controlled, and fully customisable without any vendor relationship or renewal negotiation.
